# ⚡ Game Launcher

Простой веб-лаунчер для скачивания игр через торренты. Работает на GitHub Pages без сервера.

## 🚀 Быстрый старт

### Вариант 1: GitHub Pages (Бесплатно, без сервера)

1. **Откройте в браузере:**
   ```
   https://rafman872-dev.github.io/launcher/
   ```

2. **Используйте:**
   - Введите название игры в поиск
   - Выберите фильтр (Все / Тяжелые / Легкие)
   - Нажмите "СКАЧАТЬ"
   - Откроется ваш торрент-клиент

### Вариант 2: Локально (для разработки)

```bash
# Клонируйте репозиторий
git clone https://github.com/rafman872-dev/launcher.git
cd launcher

# Откройте в браузере локально
# Способ 1: Просто откройте index.html
open index.html

# Способ 2: Используйте http-server (если установлен)
npx http-server
# Откройте http://localhost:8080
```

## 📋 Формат games.json

```json
{
  "name": "Provider Name",
  "downloads": [
    {
      "title": "Game Name",
      "fileSize": "50 GB",
      "uris": ["magnet:?xt=urn:btih:..."],
      "uploadDate": "2026-06-04T10:30:00Z"
    }
  ]
}
```

Добавляйте новые игры в этот файл.

## 🔧 Требования

- ✅ Браузер (Chrome, Firefox, Edge)
- ✅ Торрент-клиент (qBittorrent, Transmission, и т.д.)
- ❌ Никакого сервера не нужно!

---

## 📱 Использование

### Поиск
Введите название игры в строку поиска.

### Фильтры
- **Все** - все игры
- **Тяжелые** - больше 50 GB
- **Легкие** - меньше 50 GB

### Скачивание
Нажмите на кнопку "СКАЧАТЬ" - откроется магнет-ссылка в вашем торрент-клиенте.

---

## 🎮 Поддержка геймпада

Подключите геймпад и используйте:
- **D-Pad ↑/↓** - скролл по картам
- **D-Pad ←/→** - смена фильтров
- **A** - открыть ссылку
- **B** - закрыть

---

## 📝 Добавление новых игр

1. Отредактируйте `games.json`
2. Добавьте новую игру в массив `downloads`
3. Коммитьте и пушьте в GitHub

Пример:
```json
{
  "title": "New Game 2026",
  "fileSize": "60 GB",
  "uris": ["magnet:?xt=urn:btih:abc123..."],
  "uploadDate": "2026-06-04T12:00:00Z"
}
```

---

## 🌐 Развертывание

### На GitHub Pages (бесплатно)

1. Отредактируйте `games.json` с вашими играми
2. Коммитьте в `main` ветку
3. GitHub Pages автоматически обновит сайт
4. Доступен по адресу: `https://yourusername.github.io/launcher/`

### На собственном сервере

Просто скопируйте файлы на ваш хостинг:
- `index.html`
- `games.json`

Всё работает как статический сайт!
