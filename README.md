# ⚡ Game Launcher

Простой веб-лаунчер для скачивания игр через торренты. Работает на GitHub Pages без сервера.

## 🚀 Быстрый старт

### Вариант 1: GitHub Pages (Бесплатно, без сервера)

1. **Откройте в браузере:**
   ```
   https://rafman872-dev.github.io/launcher/
   ```

2. **Используйте:**
   - Введите название игры в поиск
   - Выберите фильтр (Все / Тяжелые / Легкие)
   - Нажмите "СКАЧАТЬ"
   - Откроется ваш торрент-клиент

### Вариант 2: Локально (для разработки)

```bash
# Клонируйте репозиторий
git clone https://github.com/rafman872-dev/launcher.git
cd launcher

# Откройте в браузере локально
# Способ 1: Просто откройте index.html
open index.html

# Способ 2: Используйте http-server (если установлен)
npx http-server
# Откройте http://localhost:8080
```

### Вариант 3: На приставке (Steam Deck, Linux, Windows)

**Для Steam Deck:**
```bash
# В режиме Desktop (KDE Plasma)
1. Откройте файловый менеджер
2. Перейдите в папку проекта
3. Правый клик → Открыть в терминале
4. Запустите:
   npx http-server

5. Откройте браузер (Firefox)
6. Перейдите: http://localhost:8080
7. Подключите джойстик и используйте!
```

**Для Linux / Windows:**
```bash
cd /path/to/launcher
npx http-server
# Откройте браузер: http://localhost:8080
```

**⚡ Быстро (просто файл):**
Скопируйте папку `launcher/` на приставку и откройте `index.html` двойным кликом.

## 🔧 Требования

**Минимально:**
- ✅ Браузер (Chrome, Firefox, Edge, Safari)

## 📱 Использование

### Поиск
Введите название игры в строку поиска.

### Фильтры
- **Все** - все игры
- **Тяжелые** - больше 50 GB
- **Легкие** - меньше 50 GB

### Скачивание
Нажмите на кнопку "СКАЧАТЬ" - откроется магнет-ссылка в вашем торрент-клиенте.

## 🎮 Поддержка геймпада

Подключите геймпад и используйте:
- **Left Stick ↑/↓** - скролл по картам
- **Left Stick ←/→** - смена фильтров
- **D-Pad ↑/↓** - скролл по картам
- **D-Pad ←/→** - смена фильтров
- **A / Y** - открыть ссылку
- **LT / RT** - прокрутка страницы

**📊 Отладка:** Внизу справа появится информация о подключённом джойстике.

  "fileSize": "60 GB",

  "uploadDate": "2026-06-04T12:00:00Z"

## �️ Геймпад: XInput vs D-Input

**Как это работает:**
- Браузер использует **Gamepad API** (стандарт W3C)
- Браузер автоматически конвертирует D-Input → XInput
- Поддерживаются все известные джойстики!

**Если геймпад не работает:**

1. **Переподключите джойстик** и перезагрузите браузер
2. **Проверьте в консоли** (F12 → Console):
   ```javascript
   navigator.getGamepads()[0]  // Должно вывести информацию о джойстике
   ```
3. **Попробуйте другой браузер** (Chrome работает лучше всего)
4. **На Steam Deck**: установите драйверы джойстика через `Settings → Controller`

**Если ничего не работает:**
- Используйте клавиши вместо геймпада (стрелки + Enter)
- Либо кликайте мышью / тачпадом
